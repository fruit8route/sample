≪Q≫　ASDとは？

  1. そもそもASDって何の略？
  2. どのような症状があるのか？
  3. 治療法は？

①そもそもASDって何の略？

―ASDとは「Autisim Spectrum Disorders」の頭文字を取った単語です。

  • "Autisim"=自閉症
  • "Spectrum"=連続体 (傾向の度合)
  • "Disorders"=障害

　...まとめると、自閉傾向の強弱から生じる障害という意味合いになります。(非常に大雑把ですが)
これをさらに”症状の強さ・度合い”によって段階分けすると、、、
  • アスペルガー症候群
  • 高機能自閉症
  • 自閉症

　　簡単な図解をすると以下↓の表の通りです。

　　　　　　　　　　※カナータイプ：この用語に関しての説明はここでは省きます。
　　　　　　　　　　　　　知的障害：次項目【どのような症状があるのか？】にて説明します。

②どのような症状があるのか？

―この「症状」に関しても、大きく三つに分けることができます。

ⅰ)社会性の障害
ⅱ)言語面の発達の遅れ
ⅲ)これらの症状のせいで適応に困難が生じるケース。

ⅰ)社会性の障害

　・・・一言で表せば、対人スキルを指します。

⇒通常、人間は人とのコミュニケーションにおいて、表情やしぐさ、言葉の脈絡から意図を汲み取り情報を交わします。
それを踏まえた上で、相手の反応に合わせたケースバイケースの対応(リアクション)を行い、意思疎通を図るのですが、
そこに難を抱えたものを「社会性の障害」と呼びます。

　　原因として挙げられるのは主に「心の理論取得の困難」、「前頭脳領域(主に偏桃体・紡錘状回)の異常」。
　またはそれら二つが起因した社会からのドロップアウトなどです。 しかし言うまでもなく、これらは無数の要因の一部にすぎません。
　諸説あるので、興味があればまた調べてみてください。


　　Cf-用語解説
　●心の理論：対人行為の際、相手の意図や思考、発言の意図を汲み取るための技術。
　●偏桃体：相手の表情から感情を汲み取る能力を司る前頭脳領域。
　●紡錘状回：顏の形態、各パーツの認識を行う前頭脳領域。

　…余談ですが、社会性の発達は通常「より優れたコミュニケーションの見真似」により発達していくとも言われています。
　そのため、集団からのドロップアウトは学習の機会を奪ってしまうため、より対人スキルの発達を遅らせることになるとされています。

ⅱ)言語面の発達の遅れ　

　・・・物事や情報を言語化して、認知・理解する力の発達が遅れていることを指します。
　☆これを知的な遅れ【精神遅滞・知的障害】と呼びます。

　⇒言語化する力が発達しないとどのような問題が生じる？
　　⇒・物事の意味を理解、応用する力が弱くなる。≪ケースバイケースが利かなくなる。≫
　　 　・無意識に行われてる情報処理に遅れが生じる。≪例：表情を見た際の情報入力が円滑に行われない。≫

　　　　　　...これも他に無数のケースがありますが、主にこの二点が起因して様々な症状が現れるとも言われています。

ⅲ)これらの症状のせいで適応に困難が生じるケース

　・・・ⅰ・ⅱの症状をカバーするための防衛反応として、強い固執や神経症状を引き起こす事例。
　　　⇒適応障害とも呼ばれるものです。端的に言うと、「融通の利かなさ」を指します。
　　　固執：ルーチンワークや、単一の行為・対象に過度に執着し、それに反した行為を行わざるを得ない場面で様々な身体症状を引き起こす。

　　　　普段は無意識にそれらの症状をカバーできていても、強いストレス負荷がかかった際にパニックになり表面化するケース。
　　　しかしこれも自閉症状の一つなので、スペクトラムの強弱によってこの固執傾向も大きくは差は出ます。

➡私の考えとして…

◎ⅰ～ⅲで説明した各要因は、お互いに関連・影響し合い様々な症状を引き起こしています。
　例えば上記で言うと、【適応の困難⇔社会・集団からのドロップアウト⇔対人スキルの弱さ⇔情報処理の脆弱性】といった形となっており、
　どれか一つを解決すれば社会性の能力を取り戻せるといった話ではありません。
　　
　　加えてそれらが原因となって二次障害にも繋がるところが、ASDの障害たる所以なのではないでしょうか。

③治療法は？

 ―明確な治療法というのは現状確立できていません。

　・・・一応と言いますか、治療や改善に向かう手掛かりも時折論文やニュース記事で上がっています。(薬物治療の可能性など)
　　　とはいえ残念ながら、実用化まではどれもまだ程遠いのでは私は考えています。

　　　しかしASDと併発例の多いADHDの薬物治療が現在は可能なため、
　　ADHDのせいで複雑化したASDの症状を緩和するという手法ならば今も取られています。

　　　また、強い固執や集団からのドロップアウトにより、認知特性に歪みが出やすいのもASDの特徴の一つです。
　　そのため、認知特性の面からアプローチするといった意味では「認知行動療法」も有効だと言われています。
